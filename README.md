# החופש לשחק — דף נחיתה לקורס של נטע

דף נחיתה לקורס סדנאי בהנחיית נטע: **"החופש לשחק"** — מסע משחקי, חם ונשי.

## מה יש כאן

- אתר HTML יחיד (`index.html`) — בלי תלויות, בלי בנייה
- עברית RTL מלאה
- פלטה חמה־פסטלית (אפרסק, ורד, חרדל, טרקוטה) שמתאימה לאנרגיה של נטע
- תמונה של נטע ב־`assets/netta.jpg`
- רספונסיבי למובייל
- מוכן לפריסה ב־GitHub Pages / Netlify / Vercel

## מבנה הדף

1. **Hero** — שם הקורס + תמונה + פרטים מרכזיים + כפתורי CTA
2. **על הקורס** — טקסט הקדמה
3. **שמונה משחקים** — 8 מודולים (כרטיסים צבעוניים)
4. **פרטים מעשיים** — מתי, איפה, מחיר
5. **הרשמה** — וואטסאפ + מייל + שיתוף
6. **פוטר** — ציטוט סוגר

## חובה לערוך לפני השקה

ב־`index.html`, בתחילת תגית `<script>` בתחתית, יש בלוק `CONFIG`:

```javascript
const CONFIG = {
    whatsappNumber: '972501234567',   // החליפי במספר אמיתי, עם קידומת מדינה ובלי + או 0
    emailAddress: 'netta@example.com', // החליפי במייל אמיתי
    courseName: 'החופש לשחק'           // אפשר להשאיר או לשנות
};
```

עד שתעדכני — הכפתורים יראו הודעת אזהרה במקום לעבוד.

## רשות לערוך

- **תאריכים** ופרטים מעשיים — בקטע `#details`
- **טקסט "על הקורס"** — בקטע `#about`
- **שמות 8 המודולים** — בקטע `#curriculum`
- **תמונה** — להחליף את `assets/netta.jpg` (אותו שם, אותו מיקום)
