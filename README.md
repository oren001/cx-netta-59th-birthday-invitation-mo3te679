# החופש לשחק — דף נחיתה לקורס של נטע

שש גרסאות עיצוביות שונות לקורס "החופש לשחק" של נטע, עם דף בחירה מרכזי.

## מבנה הריפו

```
├── index.html      # דף הבחירה — מציג את כל 6 הגרסאות
├── v1.html         # סטיקר אלבום (צהוב/שחור/ורוד · מסיבה)
├── v2.html         # זין פאנק (שחור/לבן/אדום · עיתון)
├── v3.html         # שער מגזין (קרם/זהב · אדיטוריאלי)
├── v4.html         # ריזוגרף (ורוד פלורסנט/ירוק · פוסטר)
├── v5.html         # פולק חם (טרקוטה/מרווה · אלגנטי)
├── v6.html         # ניאון Y2K (ציאן/מג'נטה · רטרו ווב)
└── assets/
    └── netta.jpg   # תמונת נטע
```

## ניווט

- מ־`index.html` אפשר להיכנס לכל גרסה
- בכל גרסה, סרגל עליון שחור עם **"כל הגרסאות"** + **"הקודמת"** + **"הבאה"** — מאפשר לדפדף ביניהן
- כך נטע יכולה לעבור על כולן ולהרגיש איזו "היא"

## עדכון פרטי קשר (חובה לפני השקה)

בכל אחד מקבצי `v1` עד `v6`, בתחתית הקובץ בתוך `<script>`, יש בלוק:

```javascript
const CONFIG = {
    whatsappNumber: '972501234567',
    emailAddress: 'netta@example.com',
    courseName: 'החופש לשחק'
};
```

עדכן את שלושת הערכים בכל ששת הקבצים (אפשר לעשות find-and-replace).

## פיתוח מקומי

```bash
python3 -m http.server 8000
# פתח: http://localhost:8000
```
