# הקורס של נטע — דף נחיתה

דף נחיתה אלגנטי לקורס סדנאי בהנחיית נטע: **"נוכחות, גוף ותנועה"**.

## מה יש כאן

- אתר HTML יחיד (`index.html`) — ללא תלויות, ללא בנייה, פשוט פותחים בדפדפן
- עברית עם RTL מלא
- עיצוב כהה וקולנועי המתאים לאווירת הסטודיו / מרחב התנועה
- רספונסיבי למובייל
- מוכן לפריסה ב־GitHub Pages / Netlify / Vercel

## מבנה הדף

1. **Hero** — כותרת, תיאור קצר, פרטים מרכזיים, כפתורי CTA
2. **על הקורס** — תמונה + טקסט הקדמה
3. **תוכנית הקורס** — שמונה תחנות / מודולים
4. **פרטים מעשיים** — מתי, איפה, מחיר, מה להביא
5. **הרשמה** — כפתורי וואטסאפ ומייל
6. **פוטר** — ציטוט וקרדיט

## התאמה אישית

כל הטקסטים נמצאים ישירות ב־`index.html`. ערכו אותם לפי הצורך:

### עדכון פרטי קשר
בתחילת תגית `<script>` בתחתית הקובץ:

```javascript
const WHATSAPP_NUMBER = '972501234567';   // מספר וואטסאפ עם קידומת מדינה (בלי +)
const EMAIL_ADDRESS = 'netta@example.com'; // כתובת מייל
const COURSE_NAME = 'נוכחות, גוף ותנועה';  // שם הקורס
```

### החלפת תמונת ההירו
חפשו ב־HTML את התגית `<img src="...">` בקטע `#about` והחליפו את ה־URL לתמונה אמיתית של נטע (או של אווירת הסטודיו). עדיף URL ישיר לתמונה (jpg/png), למשל מ־Imgur או מ־GitHub raw.

### עדכון תוכן
- **כותרת ראשית** — בתוך `<h1>` בקטע `#hero`
- **תיאור הקורס** — בקטע `#about`
- **שמות המודולים** — בקטע `#curriculum` (שמונה כרטיסים)
- **פרטים מעשיים** — בתוך `<ul class="details-list">` בקטע `#details`
- **תאריכים, מחיר, מיקום** — שם

### עיצוב
משתני CSS בראש ה־`<style>` (`:root`):

```css
--bg-deep: #0a0a0a;       /* רקע כהה ראשי */
--accent: #c9a96e;        /* זהב/חרדל - צבע נגינה */
--text: #ece7df;          /* טקסט עיקרי */
```

## פיתוח מקומי

פתיחה ישירה בדפדפן:
```bash
open index.html
```

או הפעלת שרת מקומי:
```bash
python3 -m http.server 8000
# פתחו: http://localhost:8000
```

## פריסה

### GitHub Pages (חינם)
1. דחפו את הקוד ל־`main`
2. Settings → Pages → Source: `main` branch
3. ההמתינו 2-3 דקות
4. הקישור: `https://USERNAME.github.io/REPO-NAME/`

### Netlify
גררו את התיקייה אל https://app.netlify.com/drop — האתר עולה תוך שניות.
